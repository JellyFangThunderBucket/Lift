# Lift

Lift is a browser userscript. Install `Main` with a userscript manager such as Tampermonkey or Violentmonkey.

## M3U8 player

The script now runs on any HTTP or HTTPS page and adds a floating **M3U8** button in the bottom-right corner.

Ways to open the player:

- Click the floating **M3U8** button.
- Press <kbd>Alt</kbd> + <kbd>M</kbd>.
- Use the userscript manager menu command named **Open Lift M3U8 Player**.

Paste an `.m3u8` URL and click **Play**. If you highlight an `.m3u8` URL before opening the player, the URL field is filled automatically.
